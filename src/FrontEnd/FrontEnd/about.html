About This Project

SmartSwitch is our 3rd year school project at Ruppin Academic Center. It was done as part of the Software Engineering course.

This project is a Full Stack application that allows a user to to make their electrical devices smart. A user may control several devices, turn them on/off and schedule timed tasks for them.
