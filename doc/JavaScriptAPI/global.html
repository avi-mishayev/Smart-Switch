addTask(task, successFunctionopt, errorFunctionopt, completeFunctionopt)

Adds a new task. Task is an object with the following properties: taskId, description, operation, deviceMac, taskType, repeatEvery and startDate.
Parameters:
Name Type Attributes Description
task object The new task object.
successFunction function <optional>
 Function to execute upon success.
errorFunction function <optional>
 Function to execute upon failure.
completeFunction function <optional>
 Function to execute upon completion.
Source:
  • SmartSwitchJavaScriptAPI.js, line 435
Example
// a task that turns a plug on every day
let newTask = {
    operation: Operations.TurnOn,
    deviceMac: "BC:DD:C2:23:D6:69",
    taskType: TaskTypes.Repeated,
    description: "Turn on every day",
    repeatEvery: 60 * 24,
    startDate: new Date()
}

// a task that will turn a plug off in 5 minutes
let d = new Date();
d.setMinutes(d.getMinutes() + 5);
let newTask = {
    operation: Operations.TurnOff,
    deviceMac: "BC:DD:C2:28:D6:60",
    description: "Turn off in 5 minutes",
    taskType: TaskTypes.OneTime,
    startDate: d
}

addTask(newTask);

getPlugSamples(mac, beginDate, endDate, successFunction, errorFunctionopt, completeFunctionopt)

Gets an array of powerUsageSample JSONs of samples between the given dates of a given plug, containing their sampleDate, current and voltage properties. The array will be ordered by date, most recent last. Each sample is for one minute following its date.
Parameters:
Name Type Attributes Description
mac string The plug's (whose samples are requested) mac address.
beginDate Date The date from which to start getting samples.
endDate Date The date from which to stop getting samples.
successFunction function Function to execute upon success.
errorFunction function <optional>
 Function to execute upon failure.
completeFunction function <optional>
 Function to execute upon completion.
Source:
  • SmartSwitchJavaScriptAPI.js, line 349
Example
let d = new Date();
d.setMinutes(d.getMinutes() - 5);
getPlugSamples("BB:DD:C2:23:D6:60", d, new Date(), function (samples) {
    for (let i in samples) {
        console.log(samples[i]);
    }
})
